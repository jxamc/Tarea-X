

**¿Por qué un AVL es mejor que un BST normal para ranking académico?
**

Un AVL es mejor porque se mantiene balanceado automáticamente.

Un BST normal puede desordenarse y volverse lento si los datos se insertan en cierto orden.

El AVL evita eso haciendo rotaciones, por lo que:

las búsquedas son más rápidas
las inserciones son más eficientes
el ranking académico se mantiene ordenado correctamente
